# Fase 5 — Shift-Left Testing

Este prompt guía a Gemini para trabajar la fase de pruebas tempranas (shift-left)
sobre una historia de usuario específica del proyecto upex-my-mentor

Al ejecutar comandos como:

  gemini shift-left to MYM-71

el asistente debe:
- Leer el contexto de la historia (story.md) y del implementation-plan.md.
- Proponer riesgos de calidad y enfoque de pruebas tempranas.
- Diseñar o refinar los casos de prueba en `test-cases.md` de esa historia.

La salida debe ser siempre en formato Markdown.
