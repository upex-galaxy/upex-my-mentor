# MVP Scope: Upex My Mentor

## 1. In Scope (Must Have) - Features Core del MVP

### EPIC-UPEX-001: Gestión de Autenticación y Perfiles de Usuario

*   **Epic Title:** Autenticación y Perfiles de Usuario
*   **User Stories:**
    *   **US 1.1:** Como usuario (estudiante o mentor), quiero registrarme con mi email y contraseña para acceder a la plataforma.
    *   **US 1.2:** Como usuario, quiero iniciar sesión con mis credenciales para acceder a mi cuenta.
    *   **US 1.3:** Como usuario, quiero poder editar mi perfil (nombre, foto, descripción, habilidades) para presentarme adecuadamente.
    *   **US 1.4:** Como mentor, quiero añadir y gestionar mis especialidades técnicas y mi tarifa por hora para que los estudiantes puedan encontrarme.
    *   **US 1.5:** Como mentor, quiero conectar mi cuenta de LinkedIn/GitHub para validar mi experiencia.

### EPIC-UPEX-002: Búsqueda y Descubrimiento de Mentores

*   **Epic Title:** Búsqueda y Descubrimiento
*   **User Stories:**
    *   **US 2.1:** Como estudiante, quiero buscar mentores por especialidad técnica (ej. React, Python, AWS) para encontrar el experto adecuado.
    *   **US 2.2:** Como estudiante, quiero filtrar mentores por precio, disponibilidad y valoraciones para refinar mi búsqueda.
    *   **US 2.3:** Como estudiante, quiero ver el perfil detallado de un mentor (experiencia, habilidades, valoraciones, tarifa) para tomar una decisión informada.

### EPIC-UPEX-003: Booking y Gestión de Sesiones

*   **Epic Title:** Booking y Gestión de Sesiones
*   **User Stories:**
    *   **US 3.1:** Como estudiante, quiero ver la disponibilidad de un mentor y reservar una sesión 1-a-1.
    *   **US 3.2:** Como mentor, quiero gestionar mi calendario y aceptar/rechazar solicitudes de sesión.
    *   **US 3.3:** Como usuario, quiero recibir notificaciones (email) de confirmación y recordatorios de mis sesiones.
    *   **US 3.4:** Como usuario, quiero acceder a un enlace de videollamada para mi sesión programada.

### EPIC-UPEX-004: Sistema de Reputación y Feedback

*   **Epic Title:** Reputación y Feedback
*   **User Stories:**
    *   **US 4.1:** Como estudiante, quiero valorar y dejar un comentario sobre mi mentor después de una sesión para compartir mi experiencia.
    *   **US 4.2:** Como mentor, quiero valorar y dejar un comentario sobre mi estudiante después de una sesión para fomentar la confianza bidireccional.
    *   **US 4.3:** Como usuario, quiero ver las valoraciones y comentarios de otros usuarios en los perfiles.

### EPIC-UPEX-005: Procesamiento de Pagos

*   **Epic Title:** Pagos y Transacciones
*   **User Stories:**
    *   **US 5.1:** Como estudiante, quiero pagar de forma segura por una sesión de mentoría.
    *   **US 5.2:** Como mentor, quiero recibir mis pagos de las sesiones completadas, descontando la comisión de la plataforma.
    *   **US 5.3:** Como usuario, quiero ver un historial de mis transacciones y pagos.

## 2. Out of Scope (Nice to Have) - Para v2+

*   **Mensajería interna:** Chat directo entre mentor y estudiante antes o después de la sesión.
*   **Integración con calendarios externos:** Sincronización automática con Google Calendar, Outlook.
*   **Sesiones grupales o talleres:** Ofrecer mentoría a múltiples estudiantes simultáneamente.
*   **Contenido educativo curado:** Recursos adicionales recomendados por mentores.
*   **Sistema de insignias/certificaciones:** Reconocimiento adicional para mentores destacados.
*   **Perfiles de empresa:** Para que empresas busquen mentores para sus equipos.
*   **Analíticas avanzadas:** Dashboards para mentores sobre su rendimiento y ganancias.
*   **Soporte multi-idioma:** Localización de la plataforma.

## 3. Success Criteria - ¿Cuándo consideramos el MVP exitoso?

*   **Criterios de Aceptación del MVP Completo:**
    *   La plataforma permite el registro y login de usuarios (estudiantes y mentores).
    *   Los mentores pueden crear y gestionar sus perfiles, incluyendo especialidades y tarifas.
    *   Los estudiantes pueden buscar, filtrar y visualizar perfiles de mentores.
    *   Se pueden reservar, gestionar y completar sesiones de mentoría 1-a-1.
    *   El sistema de pagos procesa transacciones de forma segura y distribuye los fondos correctamente.
    *   El sistema de reputación permite valoraciones bidireccionales.
    *   La plataforma es estable y funcional en navegadores modernos (Chrome, Firefox, Safari) en desktop y móvil.
*   **Métricas Mínimas a Alcanzar (en los primeros 3 meses post-lanzamiento):**
    *   **Adopción:** 50 mentores activos y 500 estudiantes registrados.
    *   **Engagement:** Al menos 100 sesiones de mentoría completadas.
    *   **Retención:** Tasa de retención de mentores >80% después de 6 meses.
    *   **Negocio:** GMV de $5,000 y $1,000 de ingresos netos.
*   **Condiciones para Lanzamiento:**
    *   Todas las épicas "In Scope" implementadas y probadas.
    *   Cero bugs críticos o de alta prioridad.
    *   Infraestructura de producción configurada y monitoreada.
    *   Plan de soporte básico establecido.
